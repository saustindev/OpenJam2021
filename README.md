#Cats Vs. Dogs:
##Fatally Furrious

Cats or Dogs? The choice is yours. Pick a side and play through this 2D shooter to earn points for your team. Show the world how much better your favorite pet is.


###Open Source Tools Used:

Audacity, Musescore, git, C++

Open Source Engine: SFML

Art and music by me, sound effects public domain.

###Gameplay Instructions:

Extract the folder and run the OpenJam2021.exe file.
